11.Truthy and Falsy Values

falsy value hay còn gọi là các giá trị giả mạo ( không trung thực )
Duới đây là các giá trị giả mạo (falsy value) 

0 , NaN, undefined , '', null
Sau khi tôi chạy code dưới đây các giá trị đều trả về false: 
      console.log(Boolean(0)); 
      console.log(Boolean(undefined));
      console.log(Boolean(''));
      console.log(Boolean(null));
Tại sao lại trả về false ?
Trả lời : trả về false vì đều chứa các giá trị không trung thực

Tiếp tục với 1 ví dụ khác : 

const myMoney = 0;

if (myMoney) {
  console.log('Bạn đã có tiền, đừng tiêu nó hết nhé');
} else {
  console.log('Bạn nghèo vl, chả nổi 1 đồng trong người');
}
Kết quả : 
Bạn nghèo vl, chả nổi 1 đồng trong người

Tại sao lại có kết quả này ?

Vì khi ta sử dụng Boolean(0) sẽ trả về false

và với đoạn code trên thì ta hiểu nó như sau : 
      if(true){
        ... thực thi code nếu điều kiện là true
      } else {
        ... thực thi code nếu điều kiện khác true
      }

Đó là kết quả vì console.log lại in ra Bạn nghèo vl, chả nổi 1 đồng trong người
